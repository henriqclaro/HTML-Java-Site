Hello World

O básico das linguagens de programação. Este é um exemplo da sintaxe básica de Java, com a classe Main, o método main e a função System.out.println(). 

                    public class Main {
    public static void main(String[] args) {
    System.out.println("Hello World");
    }
}

Tipos de Dados

Em Java, um dado precisa de uma declaração contendo o seu tipo. Cada tipo de dado tem um tamanho específico e um intervalo de valores que pode armazenar. 

                        public class Main {
    public static void main(String[] args) {
        byte oitoBits = 127; // Inteiros de -128 a 127
        short pequenoValor = 27748; // Inteiros de -32768 a 32767
        int inteiro = 1; // Inteiros de -2147483648 a 2147483647
        long longoValor = 123456789L; // Inteiros de -9223372036854775808 a 9223372036854775807
        float flutuante = 10.5f; // Racionais com 6 a 7 dígitos decimais
        double duploFlutuante = 12.59d; // Racionais com 15 a 16 dígitos decimais
        char caractere = 'A'; // Caractere único ou valores ASCII
        boolean booleano = true; // Valores booleanos (verdadeiro ou falso)
        String texto = "Hello World"; // Pedaço de texto, tipo de dado não primitivo
    }
}

Dos dados acima, apenas o String não é um tipo de dado primitivo.

Impressão de Variáveis

As variáveis em Java podem ser impressas na tela utilizando a função System.out.println() e o símbolo '+', que concatena o conteúdo em uma única String impressa. 

                    public class Main {
    public static void main(String[] args) {
        String nome = "Erick";
        int idade = 34;
        System.out.println("O " + nome + " tem " + idade + " anos.");
    }
}

Output: "O Erick tem 34 anos."
