Hello World

O básico das linguagens de programação. Este é um exemplo da sintaxe básica de Java, com a classe Main, o método main e a função System.out.println(). 

public class Main {
   public static void main (String[] args) {
       System.out.println("Hello World");
   }
}

Output:
> Hello World

Tipos de Dados

Em Java, um dado precisa de uma declaração contendo o seu tipo. Cada tipo de dado tem um tamanho específico e um intervalo de valores que pode armazenar. 

byte oitoBits = 127;    //Inteiros de -128 a 127
short pequenoValor = 27748;    //Inteiros de -32768 a 32767
int inteiro = 1;    //Inteiros de -2147483648 a 2147483647
long longoValor = 123456789L;    //Inteiros de -9223372036854775808 a 9223372036854775807
float flutuante = 10.5f;    //Racionais com 6 a 7 dígitos decimais
double duploFlutuante = 12.59d;    //Racionais com 15 a 16 dígitos decimais
char caractere = 'A';   //Caractere único ou valores Unicode
boolean booleano = true;    //Valores booleanos (verdadeiro ou falso)
String texto = "Hello World";   //Pedaço de texto, tipo de dado não primitivo

Dos dados acima, apenas String é um tipo de dado não primitivo.

Impressão de Variáveis

As variáveis em Java podem ser impressas na tela utilizando a função System.out.println() e o símbolo '+', que concatena o conteúdo em uma única String impressa. 

String nome = "Erick";
int idade = 34;
System.out.println("O " + nome + " tem " + idade + " anos.");
System.out.printf("O %s tem %d anos.\n", nome, idade);  //Impressão formatada

Output:
> O Erick tem 34 anos.
> O Erick tem 34 anos.

Métodos String

Em Java, o tipo String pode ser manipulado com alguns métodos específicos.


O método length() retorna o número de caracteres de uma String.

String nome = "Erick";
System.out.println(nome.length());

Output:
> 5


O método charAt() retorna o caractere de uma String na posição especificada.
O primeiro caractere de uma String é o índice 0, o segundo é o índice 1 e assim por diante.
O último caractere de uma String é o índice length() - 1.

String nome = "Erick";
System.out.println(nome.charAt(0));

Output:
> E


O método toUpperCase() retorna uma String em letras maiúsculas.
O método toLowerCase() retorna uma String em letras minúsculas.
O método trim() remove os espaços em branco no início e no final de uma String.

String nome = "   Erick   ";
System.out.println(nome.trim().toUpperCase());
System.out.println(nome.trim().toLowerCase());
System.out.println(nome.trim());

Output:
> ERICK
> erick
> Erick


O método replace() substitui um caractere por outro em uma String.
O método substring() retorna uma parte de uma String, especificando o índice inicial (inclusivo) e o índice final (exclusivo).
O método indexOf() retorna o índice da primeira ocorrência de um caractere em uma String.

String nome = "Erick";
System.out.println(nome.replace("E", "e"));
System.out.println(nome.substring(1, 3));
System.out.println(nome.indexOf("i"));

Output:
> erick
> ri
> 2


O método equals() compara duas Strings.
O método equalsIgnoreCase() compara duas Strings, ignorando maiúsculas e minúsculas.

String nome1 = "Erick";
String nome2 = "erick";
System.out.println(nome1.equals(nome2));
System.out.println(nome1.equalsIgnoreCase(nome2));

Output:
> false
> true

Laços de Repetição

Os loops são estruturas de repetição que permitem executar um bloco de código várias vezes.
Existem três tipos principais de loops em Java: for, while e do while.


O loop for é usado quando o número de iterações é conhecido.

for (int i = 0; i < 5; i++) {
   System.out.println(i);
}

Output:
> 0
> 1
> 2
> 3
> 4


O loop while é usado quando o número de iterações não é conhecido.
O loop continua enquanto a condição for verdadeira.
O loop do while é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez.

int i = 0;
while (i < 5) {
   System.out.println(i);
   i++;
}

Output:
> 0
> 1
> 2
> 3
> 4


O loop for each é usado para iterar sobre elementos de uma coleção, como arrays.
Ele é útil quando você não precisa do índice do elemento.

String[] nomes = {"Erick", "João", "Heitor"};
for (String nome : nomes) {
   System.out.println(nome);
}

Output:
> Erick
> João
> Heitor


Os comandos break e continue são usados para controlar o fluxo dos loops.
O comando break encerra o loop imediatamente.
O comando continue pula a iteração atual e continua com a próxima.

for (int i = 0; i < 5; i++) {
   if (i == 4) {
       break;
   }
   if (i == 2) {
       continue;
   }
   System.out.println(i);
}

Output:
> 0
> 1
> 3
